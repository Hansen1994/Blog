一、javascript数据类型

Javascript属于若语言类型，只有一个唯一变量标示符“var”，但内部也区分数据类型,

PS：Javascript中的基本数据类型有如下:字符串、数字、布尔、数组、对象、Null、Undefined;

var x = "xxx";//字符串(string)

var num1 = 2;//整数类型(int)

var num2 = 2.0;//浮点数类型(floor)

var flag = false;//boolean

var flag = true;//boolean

var arr = ["xxx","xxuuu","zzz"];//数组类型，Array

var n1 = null;//null类型

var n2 = undefined;//(找不到)

二、注意事项

1.var变量的命名与java相同，以字母，下划线_ ,$美元符号开头

2.变量名定义一定要有意义。（PS：毫无意义的命名只会影响代码的可读性）

3.Javascript中的方法函数function(){ }，也属于对象类型。

4.格式是:function 方法名(Arguments){语句}

5.调用方法，方法名后要加（this);指向当前方法



var nchar="我是字符哦！记住哦，只有在双引号或者是单引号之间的才是字符哦";
