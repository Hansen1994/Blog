6.box-shadow和text-shadow阴影的用法

(1)box-shadow

	text-shadow: x-shadow y-shadow blur spread color inset;
	x和y的方向根据平面直角坐标系。就是说：南为y，东为x
	

		

			x-shadow
			必需
			水平阴影的位置。允许负值。
		

		

			y-shadow
			必需
			垂直阴影的位置。允许负值。
		

		

			blur
			可选
			模糊距离。
		

		

			spread
			可选
			阴影的尺寸
		

		

			color
			可选
			阴影的颜色。请参阅 CSS 颜色值
		

		

			inset
			可选
			改为内部阴影，省略则不用设置
