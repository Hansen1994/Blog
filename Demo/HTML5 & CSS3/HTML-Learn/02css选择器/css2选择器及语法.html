6.css样式 - CSS 类选择器

.center{} <div class="center"> 
		

		1.在 CSS 中，类选择器以一个点号显示
		2.h1 和 p 元素都有 center 类。这意味着两者都将遵守 ".center" 选择器中的规则

7.css样式 - id 选择器

#w1{} <div id="w1"> 
		

			1.id 选择器可以为标有特定 id 的 HTML 元素指定特定的样
			2.id 选择器以 "#" 来定义
			3.id 属性只能在每个 HTML 文档中出现一次
			4.在现代布局中，id 选择器常常用于建立派生选择器
