定位position


1.fixed 固定定位

位置固定在窗口某个位置，不管滚动条如何拖动，都不动 （left right 水平方向位置)(top bottom 竖直方向的位置)

比如像一些网站右下角的返回头部按钮我是fixed



2.relative 相对定位

改变元素的位置，但是还是暂着其原来的位置不会变

如果对一个元素进行相对定位，它将出现在它所在的位置上。然后，可以通过设置垂直或水平位置，让这个元素“相对于”它的起点进行移动。

代码案例:

我是relative定位哦

代码： .relative{width:100px;height:100px;background:#6CC;margin-left:300px;position:relative;top:20px;left:20px;}



3.absolute 绝对定位

以相对定位元素，为参考，改变位置，但是不占局原来的位置

位置可以任意的改变，并且不占空间，相当于悬浮在当前块的上面

代码案例

直接写代码，是根据浏览器为相对定位，所以此代码不给出，在下面的绝对与相对定位给出

4.相对定位与绝对定位

	1、绝对定位元素一定是和相对定位元素一起
	2、如果没定位元素，则相对于body 窗口定位
	3、绝对定位的元素一定要在相对定位的元素的里面
	4、绝对定位的元素的位置相对于最近的已定位祖先元素

代码案例

		

		代码展示
				.guo{width:100px;height:100px;background:#696;position:relative;}
				.xian{width:25px;height:25px;background:#F06;position:absolute;top:50px;left:50px;}


5.z-index的知识

①z-index一定要配合绝对定位与相对定位使用

②他的值可以为正也可以为负 （正值 : z-index:3; 负值： z-index: -1; ），切记，正值越大越上面，负值越小越上面

用的最多实在导航方面



Static：保持文档流 ;

Relative：相对本身的原始位置发生位移且保持文档流，占空间 ;

Absolute：脱离文档流，不占空间且相对于其包含块来定位 ;
