七星鲁王宫

　“双鱼玉佩？”收到这个消息时我有些愕然，十年前的事件被再次提起，不知是福是祸，可毕竟是九门发布的任务，倒也不好推辞。　　手机铃声响起，是黑眼镜打来的，难道他也接到消息了？我快步走到窗边，按下接听键。　　“我想你也收到了。”忽闻黑眼镜如此正经的语气，我微微一愣，不禁也正色几分，低声回应道：“嗯。”　　电话那头陷入了沉默，准确来说应该是有人捂住了话筒，只能隐约听到些朦胧的交谈声，黑眼镜身边还有别人吗？　　“我和哑巴明天去你那。”正当我等得不耐烦时，黑眼镜的声音恰巧传来，“还有，有个新朋友介绍给你。”　　我应了一声，挂断电话，推开窗，点上烟，清醒一下混沌的大脑，暗叹自己的习惯越来越像老干部了。　　新朋友？不知道是什么样的人，总归是黑眼镜介绍来的，应该可以信任吧。想来倒是很久没见过闷油瓶了，毕竟他总是行踪无觅的。　　我掐灭烟，回房间打开电脑，开始搜索有关双鱼玉佩的资料。　　粗略地看了一遍，什么镜像反物质、超十一维、51区、活死人等等细思极恐的词汇都与其有关，之中人们关注度最高的当属彭加木失踪一案了，此次九门派发的任务多半与他有关。　　我满怀心事，辗转反侧，第二天只好顶着两只熊猫眼去见闷油瓶他们，免不了被黑眼镜打趣一番，还被坑去一顿“楼外楼豪华餐”。　　我们几个静静地吃着一桌子菜，气氛极为尴尬诡异，黑眼镜不知是什么情况，如今闷油瓶更是让我看不透了，他仿佛随时可以毫无牵挂地从这个世界抽离出去，这种认知让我非常不舒服。　　另外还要说的就是同他们一起来的陌生女子，阿淼。她看起来跟我差不多大，性子有些冷淡，不太愿与人接触，她和闷油瓶坐在一起感觉正个包间都凉快了不少，难道黑眼镜专门喜欢跟这种类型的人一起待着？　　实在忍受不了这恼人的氛围，我转头看向黑眼镜：“所以这次该怎么弄？”　　“怎么弄？该怎么弄怎么弄。”黑眼镜豪气地端起酒杯一饮而尽，“把人都叫一叫，召集一下，九门这次肯定给不少人发了任务，大家一起行动总好一点，谁知道他们藏着什么好东西，那群人一肚子坏水。”说完，他又给自己满上一杯，“这儿的就不错啊，一会弄几瓶带着。”　　“行，回头我问问小花他们。”有他和闷油瓶坐镇，我顿时踏实许多。　　我趁把他们送去旅店的功夫给小花等人打了电话，才知道人家早有安排，我们大概是最晚得到消息的。“小花他们已经出发了，胖子也去了，我跟他约好在罗布泊镇汇合后一起行动。”我将通话内容转述给三人。　　“呵，就知道那群人肯定迫不及待，这玉佩可伴着不少值钱的东西。”黑眼镜懒懒地吐出个烟圈，语气多少有些不屑的意味，不知道他对九门的态度因何而起，可能是以前被九门坑过吧。　　“有今晚的巴士，凌晨到，再来就是后天的了。”阿淼突然出声，我还在猜他方才一直拿着手机捣鼓什么，原来是在查车次。不过后天的肯定不行，等到那时黄花菜都凉了，只能乘今晚的。我抬腕看了下时间，有点赶啊。　　黑眼镜随手将烟丢进垃圾桶，也没有管其有没有熄灭，嘴角微勾：“马上添置装备，阿淼先把票买了。”　　～～～　　卧铺客车的晃动将我从睡梦中拉回，仰躺着揉了揉眼睛，车内只有两侧开着的昏暗的灯光，估摸估摸应该快到了，坐起身，瞧见闷油瓶正望着窗外发呆。　　话说这家伙降低存在感的技能又升级了，要不是我一路注意着，差点以为他又偷跑掉了。　　四下看看，黑眼镜正玩着刀，阿淼在看手机，好吧，四个人就属我睡得最没心没肺。　　刚穿好鞋子，客车的大喇叭就响了起来：亲爱的旅客朋友们，罗布泊镇站到了，请各位有序下车，注意安全，多谢合作。　　随着人群下了车，猛然踩在这软绵的沙地上还真有些不习惯，回头正撞见阿淼一个趔趄险些摔倒在地，我不厚道地笑出声来，换来她一记白眼。　　与阿淼混熟之后发现她还是挺有人情味的，好在不是一个女版闷油瓶。　　“你们这是想困死胖爷我啊。”肩头突然被重重地拍了一下，吓得我打了个激灵，回头便看见胖子那张熟悉的大脸，“呦，这位是？”他偏过头看向阿淼。　　黑眼镜嘬了口烟：“介绍一下，阿淼，别看是女生，身手可不差。”　　“幸会幸会，相互照顾照顾啊。”见阿淼只是淡淡地点点头，胖子也没自讨无趣，跑去跟刚走的下来的闷油瓶打了招呼，回头对我说道：“还好你胖爷有先见之明，接到电话便给你们订了房间，现在那火爆着呢。”　　“怎么回事？”我有些疑惑，这旅店的规模算不上大但也起码是中等，况且现在是旅游淡季，怎么会出现这种状况？　　只见胖子一脸恨铁不成钢地拍了拍我：“你以为就咱们几个对那东西感兴趣啊，我听你那花儿爷说，九门一个月前就派人去过，没一个活着回来的，这次已经是第二队了，咱们几个也就算个备胎。”他顿了顿，“还有啊，九门给不少人透露了消息，愿意出重金请他们帮忙，也不知道为什么如此执着。”　　听完胖子的话，我皱了皱眉，看来这次已经不是单纯盗个斗这么简单了，最重要的在意的便是九门的目的若能知晓他们此举的原因，之后大概会好办许多，至少没现在这么被动。　　“你们两个别卿卿我我的了，养足精神，明天好出发。”黑眼镜的喊声将我的思绪拉回，我和胖子应了一声，前后脚进入旅店，今晚注定又是个不眠之夜啊。
