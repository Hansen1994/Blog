3.多选按钮type="checkbox"

爱好多选：
爱好多选
<input type="checkbox" name="basketball" id="lq"/><label for="lq">篮球</label>
<input type="checkbox" name="football" id="zq"/><label for="zq">足球</label>
<input type="checkbox" name="volletball" id="pq"/><label for="pq">排球</label>
*checked="checked"为默认选择为此项
*label 元素不会向用户呈现任何特殊效果。不过，它为鼠标用户改进了可用性。如果您在 label 元素内点击文本，就会触发此控件。
就是说，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上<label> 标签的 for 属性应当与相关元素的 id 属性相同.
*name可以不同或者不写

3.一些按钮



<form action="http://www.baidu.com/baidu" method="">
<input type="text" name="no"/>
<input type="submit" name="na" value="提交按钮"/>
<input type="button" name="nb" value="普通按钮"/>
<input type="reset" name="nc" value="重置按钮"/>
<input type="file" name="nd" value="浏览文件"/>
<input type="hidden" name="hid" value="value"/>
</form/>
action:当用户单击确认按钮时，表单的内容会被传送到另一个文件。 表单的动作属性定义了目的文件的文件名。由动作属性定义的这个文件通常会对接收到的输入数据进行相关的处理。
method:method属性有post和get两种方式；get是把参数数据队列加到提交表单的ACTION属性所指的URL中，值和表单内各个字段一一对应，在URL中可以看到。 post是通过HTTP post机制，将表单内各个字段与其内容放置在HTML HEADER内一起传送到ACTION属性所指的URL地址。用户看不到这个过程。
*如果要提交给后台必须写上form
*hidden内容隐藏不显示，主要是用于提交数据给后台
