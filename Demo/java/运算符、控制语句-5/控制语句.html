例题展示
问题：
1. 系统给定人的年龄【1 - 100】
2. 通过人的年龄，来判断你是那个阶段的人 ！
[1-19] 我是青少年哦
[20-35]正在壮年时期，好好努力哦
[36-55]我老爸就在这个年龄段哦
[56-100] 回望人生
public class IfTest01{
public static void main(String[] args){
int age = 40;
if(age >= 1 && age <= 19){
System.out.println("我是青少年哦");
} else if(age >= 20 && age <= 35){
System.out.println("正在壮年时期，好好努力哦");
} else if(age >= 36 && age <= 55){
System.out.println("我老爸就在这个年龄段哦");
} else if(age >= 56 && age <= 100){
System.out.println("回望人生");
}
}
}
输出：我老爸就在这个年龄段哦




❤switch 语句

	语法：	
 switch(int类型){
    case int类型:
       java语句;
       java语句;
       break;

   case int类型:
       java语句;
       java语句;
      break;

   case int类型:
      java语句;
      java语句;
      break;

   default:
      java语句;
                }
public class SwitchTest{
public static void main(String[] arge){
char num = 'B';
switch(num){
case 'B':
System.out.println("做人要行善，多做好事");
break;
case 'C':
System.out.println("道之道，非常道");
break;
case 'D':
System.out.println("明可明，非常明");
break;
default:
System.out.println("神明自在功德心！");
}
}
}

switch后边的括号里边可以填写 byte/short/char类型，因为他们可以自动转换
break; 语句是可以没有的
default 语句也是可以没有的

※switch 的break问题

case 是可以穿透的 直到出现break;语句时 就跳出,没有break;语句 就直接穿透

如果 default放到最前边 从上到下执行,知道出现break结束跳出

从上到下匹配case，条件不满足 最后执行default

public class SwitchTest{
public static void main(String[] arge){
char num = 'B';
switch(num){
case 'B':
System.out.println("做人要行善，多做好事");

case 'C':
System.out.println("道之道，非常道");
break;

case 'D':
System.out.println("明可明，非常明");
break;

default:
System.out.println("神明自在功德心！");

}
}
}

输出结果：
做人要行善，多做好事
道之道，非常道
