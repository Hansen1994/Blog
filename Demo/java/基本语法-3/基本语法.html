1.Java标识符

Java所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符。

注：
		1. 所有的标识符都应该以字母（A-Z或者a-z）,美元符（$）、或者下划线（_）开始
		2. 首字符之后可以是任何字符的组合
		3. 关键字不能用作标识符
		4. 标识符是大小写敏感的
String _UserName = "hello";
String $username = "good";
System.out.println("用户名："+_UserName+" "+$username);
