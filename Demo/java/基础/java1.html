第一个java demo

			public class num{
				public static void main(String[] args){		
				System.out.println("今天是个不错的日子");
				}
			}
		
	1.打开UltraEdit，把上面的代码添加进去；
	2.把文件名保存为：num .java；
	3.打开cmd命令窗口，进入目标文件所在的位置，假设是C:\
	4.在命令行窗口键入 javac num.java  按下enter键编译代码。如果代码没有错误，cmd命令提示符会进入下一行。（假设环境变量都设置好了）。
	5.再键入java num 按下Enter键就可以运行程
