1.方法初步-什么是方法-方法的调用

方法是一段代码的片段，这个片段的代码可以完成特定的功能，并且可以重复利用


自定义一个方法,完成整数的求和
public class Ffchu{

//先定义成一个 public static

public static void mySum(int a,int b){
int c=a*b;
System.out.println("c="+c);
}

//入口

public static void main(String[] args){

//加有static的方法，调用的时候必须采用“类名.” 的方式调用

Ffchu.mySum(10,10);
}

}

输出结果：100


2.方法定义语法-有无形参-return返回值

	定义方法的语法：
		
		[方法修饰列表] 返回值类型  方法名（方法参数列表）{
				
			方法体
		
		}
		
		1. 方法修饰列表是可以选项，现在暂时先写成 public static
		2. 返回值类型 可以是java中的任意类型（基本数据类型，引用数据类型）
		3. 如果该方法执行后，没有任何返回值 就用 void
		4. 方法名只要是合法的标识符就可以的
		5. 方法的形式参数列表，可以有参数，也可以没有参数，如果有多个用“,”逗号隔开
public class MethodTest03{

// public static 修饰符

// void 无返回值

// arry 方法名

// 没有形式参数

public static void arry(){
System.out.println("我是最帅气的Arry");
}

// public static 修饰符

// void 无返回值

// arry1 方法名

// int a,boolean b 是形式参数列表

public static void arry1(int a,boolean b){
int i = 0;
if(b){
i = a;
}
System.out.println("i = "+i);
}

// 该方法有返回值，程序必须使用return 语句返回

public static int arry2(int a,int b){
int c = a + b;
return c;

// 程序执行到return 这行，arry2这个方法就执行结束，并返回值c

// return语句下边不能再有其他的代码,因为根本就不能执行到,也不能编译通过

} public static void main(String[] args){
MethodTest03.arry();
MethodTest03.arry1(10,true);
System.out.println("num = " + MethodTest03.arry2(20,50));
}
}

5.方法的重载

	缺点:
	1.  程序开发者需要记忆更多的方法名
	2.	 程序代码不美观（优雅）。

比如:

public class MethodTest05

		
// 主方法

	public static void main(String[] args){	

		
// 对于程序来说，需要记忆三个不同的方法名
	
		System.out.println("int类型："+Arry2.sumInt(10,20));

		System.out.println("double类型："+Arry2.sumDouble(1.0,2.0));
		System.out.println("long类型："+Arry2.sumLong(10L,20L));		
	}	
}

class Arry2

	public static int sumInt(int a,int b){
		return a + b;	
	}
	public static double sumDouble(double a,double b){
		return a + b;	
	}
	public static long sumLong(long a,long b){
		return a + b;	
	}
}



	什么时候需要方法重载？
	
	1. 发生在同一类中
	2. 方法名相同
	3. 参数列表不同（类型、个数、顺序）
	4. 和返回值类型无关
	5. 和方法的修饰符也无关

例如：

public class MethodTest06{
	
// 主方法

	public static void main(String[] args){		
		
// 对于程序来说，需要记忆三个不同的方法名,而都改成一个就方便了

		System.out.println("int类型："+Arry6.sum(10,20));
		System.out.println("double类型："+Arry6.sum(1.0,2.0));
		System.out.println("long类型："+Arry6.sum(10L,20L));	
	}
}
class Arry6{
	public static int sum(int a,int b){
		return a - b;	
	}	
	public static double sum(double a,double b){
		return a * b;	
	}
	public static long sum(long a,long b){
		return a + b;	
	}
