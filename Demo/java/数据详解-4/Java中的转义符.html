1.制表符

\ 斜线在Java语言中的转移功能，可以将普通t字符，转换成“制表符”

System.out.println("i = " + i);

System.out.println("abc");

2/tab键

\t相当于电脑的键盘的tab键

System.out.print("j = " + j);

System.out.println("今天又是奋斗的一天 ！");

3.单引号

char k = '\'';

System.out.println("k = "+k);

输出结果：'

4.斜线

char l = '\\';

System.out.println("l = "+l);

输出结果：\

5.双引号

char n = '\"';

System.out.println("n = "+n);

输出结果："

6.换行符

char m = '\n';

 // 相当于键盘回车键

System.out.println("");

// print 不换行，println 换行

System.out.print("今天有是个奋斗的日子 ！");
