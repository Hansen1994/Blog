一、对象的使用范围和运用

Javascript除了 数字，字符串，布尔值（true/false），null和undefined值。

其他的所有值都是对象。如：数组，object，function,正则表达式Regex,日期Date。

语法:
对象 = {属性：值}
案例:
var emobj = {}; /*空对象*/
var book = {"name":"三毛"}；
var book = {"name":"三毛","date":"2015-1-19"}；/*对象属性可以多个用英文的逗号间隔*/

二、对象的2种检索修改方式

方式1:

var people = {"name":"步枪","ages":"2"};

document.write("名字:"+people.name+"-----年龄:"+people.ages);

方式2:

var people = {"name":"步枪","ages":"2"};

document.write("名字:"+people["name"]+"-----年龄:"+people["ages"]);

打印结果都为:名字:步枪-----年龄:2
空对象直接添加属性：book["price"] = 100;

三、对象的更新

var people = {"name":"步枪","ages":"2"};

people.name = "小米";

people.ages = "5";

document.write("新名字:"+people.name+"-----新年龄:"+people.ages);

五、对象的创建

var str = {};

str.name = "冠军";

str.ages = "25";

document.write("名字:"+str.name+"-----年龄："+str.ages);

名字冠军-----年龄25

六、对象的删除

var people = {name:"沧海"};

document.write("没有删除前的："+people.name);

delete people.name;

document.write("删除后的："+people.name);

没有删除前的：沧海
删除后的：undefined

七、对象的获取方式for、in

var classroom = {room:"教室一",number:48,code:"13210132"};

for (key in classroom)

{

document.write("
key的值是:"+key+"------classroom的值是:"+classroom[key]);

}

key的值是:room------classroom的值是:教室一
key的值是:number------classroom的值是:48
key的值是:code------classroom的值是:13210132
