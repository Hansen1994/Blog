JS中的Doucment-Dom

当网页被加载时，浏览器会创建页面的文档对象模型（Document Object Model）

HTML DOM 模型被构造为对象的树，HTML DOM 树

JavaScript 能够改变页面中的所有 HTML 元素、HTML 属性、CSS 样式、

JavaScript 能够对页面中的所有事件做出反应

一、Document对象属性

document.title 设置文档标题等价于HTML的title标签

(选第一个作为案例:标题已变，请看上边)

document.bgColor 设置页面背景色

document.fgColor 设置前景色(文本颜色)

document.linkColor 未点击过的链接颜色

document.alinkColor 激活链接(焦点在此链接上)的颜色

document.vlinkColor 已点击过的链接颜色

document.URL 设置URL属性从而在同一窗口打开另一网页

document.fileCreatedDate 文件建立日期，只读属性

document.fileModifiedDate 文件修改日期，只读属性

document.fileSize 文件大小，只读属性

document.cookie 设置和读出cookie

document.charset 设置字符集 简体中文:gb2312

二、 常用对象方法

document.write() 动态向页面写入内容

document.createElement(Tag) 创建一个html标签对象

document.getElementById(ID) 获得指定ID值的对象

document.getElementsByName(name) 获得指定name值的对象

document.getElementsByTagName(tagName) 获得指定标签值的对象

document.body.appendChild(oTag) 相对父标签，添加子标签元素。

document.getElementById(”ID”).innerText 动态输出文本

document.getElementById(”ID”).innerHTML 动态输出HTML

document.getElementById(”ID”).outerText 同innerText

document.getElementById(”ID”).outerHTML 同innerHTML

注意:

①Javascript中，写方法和定义的时候严格区分大小写

②方法：document.getElementById(元素的id)根据id获取对象

③当页面上同时存在两个相同的ID时，只会打印第一个id对应的值

④.value只有form元素才有意义

⑤id选择器:唯一标识符

通过document.getElementsByName(name)举例

document.get（获取）Elements(元素) By(通过) TagName(标签)(tagName)通过

<input type="checkbox" name="hobbys" value="爬山">爬山

<input type="checkbox" name="hobbys" value="音乐">音乐

<input type="checkbox" name="hobbys" value="读书">读书

<input type="checkbox" name="hobbys" value="游泳">游泳

<script type="text/javascript">

var divObj = document.getElementsByName("hobbys");

divObj[2].value;

alert(divObj[2].value);

</script>

输出结果:

睡觉

三、innerText与innerHTML的区别:

1.innerHTML:带有格式的文本,如果标签内有嵌入标签，innerHTML，会获取除了id指定的HTML标签外，其他嵌入的标签会保留显示

2.innerText:只获取标签内的文本内容，不包含标签。

3.与innerHTML，innerText相对的outerHTML，outerText。

代码展示:

<div id="k_div"><span>今天</span>是个不错的日子</div>

<div id="k_p"><span>今天</span>是个不错的日子</div>

<script type="text/javascript">

var value1 = document.getElementById("k_div").innerHTML;

alert(value1);

var value2 = document.getElementById("k_p").innerText;

alert(value2);

</script>

打印结果:

<span>今天</span>是个不错的日子

今天是个不错的日子

注意:

①outerHTML会显示包含id的标签本身。这点与innerHTML有些许差别。

②outerText与innerText相同，输出没有差别

③Form元素用innerHTML/Text，outerHTML/Text无效（无意义）
